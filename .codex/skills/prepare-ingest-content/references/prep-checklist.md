# Preparation Checklist

Run this checklist before finalizing ingest content.

## Signal Quality

- Keep only durable context with future retrieval value.
- Remove duplicated points and conversational filler.
- Split mixed topics into separate files.

## Safety

- Remove passwords, API keys, tokens, and credential material.
- Minimize unnecessary personal data.
- Exclude sensitive details that are not needed for agent performance.

## Truthfulness

- Keep claims specific and source-grounded.
- Mark unknowns or uncertain items as `Unverified`.
- Avoid inference phrasing that implies certainty.

## Structure

- Ensure headings are clear and stable.
- Include Summary, Facts, Decisions, Constraints, Open Loops, and Suggested Tags.
- Keep output focused and concise.

## File Hygiene

- Use UTF-8 Markdown.
- Prefer filename pattern `YYYY-MM-DD-topic-brief.md`.
- Keep each file to one coherent topic.
